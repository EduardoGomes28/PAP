# 📦 PAP --- Projeto Final de Curso

Este repositório contém o **Projeto Final de Curso** denominado **YXE!**,
desenvolvido por *Eduardo Gomes*.\
O objetivo principal deste projeto é criar uma aplicação funcional que
inclui um sistema de loja com autenticação de utilizadores e gestão de
dados.

------------------------------------------------------------------------

## 🧠 Sobre o Projeto

O **YXE!** é uma aplicação que integra funcionalidades de acesso de
utilizador e operações básicas de loja.\
O projeto inclui um backend em Python e uma interface em HTML/CSS,
demonstrando conhecimentos de desenvolvimento web e integração com bases
de dados.

------------------------------------------------------------------------

## 📁 Estrutura do Repositório

  Caminho                  Descrição
  ------------------------ ------------------------------------------
  `app.py`                 Código principal da aplicação
  `verifica_usuários.py`   Script de verificação de login
  `loja.db`                Base de dados principal da loja
  `users.db`               Base de dados de utilizadores
  `static/`                Ficheiros estáticos (CSS, imagens, etc.)
  `templates/`             Modelos HTML

------------------------------------------------------------------------

## 🚀 Como Executar

### 🔧 Pré‑requisitos

-   Python 3.x instalado

### ▶️ Passos

1.  Clonar o repositório:

    git clone https://github.com/EduardoGomes28/PAP.git cd PAP

2.  (Opcional) Criar ambiente virtual:

    python -m venv venv venv`\Scripts`{=tex}`\activate  `{=tex}\#
    Windows source venv/bin/activate \# macOS/Linux

3.  Instalar dependências (caso exista requirements.txt):

    pip install -r requirements.txt

4.  Executar a aplicação:

    python app.py

A aplicação deverá iniciar em: http://localhost:5000
